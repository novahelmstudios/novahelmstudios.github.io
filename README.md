# NovaHelm Studios Website

Official website for NovaHelm Studios, built with Next.js, TypeScript, and Tailwind CSS.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

This repository is configured for GitHub Pages using the workflow in `.github/workflows/nextjs.yml`.

The site is published from the `main` branch and builds a static export to `out/`.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
